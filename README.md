## 🚀 Project Overview

This project was developed as part of the **CrowdWisdomTrading Quantitative Data Scientist Intern Assessment**. The objective was to build an end-to-end quantitative research pipeline that collects market data, engineers predictive features, trains machine learning models, backtests trading strategies, and provides an interactive dashboard for visualization.

---

## ✨ Key Features

- 📊 Automated financial data collection
- 🔄 Data preprocessing and feature engineering
- 🤖 Machine Learning model training
- 📈 Strategy backtesting
- 📉 Performance evaluation metrics
- 📊 Interactive dashboard
- 📁 Modular and scalable project structure
- ⚡ Fast and reproducible workflow

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AiErAshutoshSingh/CrowdWisdomTrading-Quant-Intern.git
```

Move into the project

```bash
cd CrowdWisdomTrading-Quant-Intern
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
